This repository includes files for running a 4 wheeled bot in a gazebo simulation.
- It can be run in gazebo and visualized through rviz2.
- Uses zed-ros2-wrapper for Zed Camera features that can also be simulated in gazebo and visualized in rviz.
- Uses Skid steer drive.
- No friction added to the bot yet.
- Can be used with teleop_twist_keyboard for controlling.

Demonstration video will be uploaded soon.
